Configurar Sync
Configure o Firebase uma vez. Depois é só abrir e usar.
Passo a passo — 3 minutos
1
Acesse console.firebase.google.com e clique em "Criar projeto" (pode usar a conta Google da agência)
2
Dê um nome (ex: popp-tracker), desative o Google Analytics e clique em "Criar projeto"
3
No menu lateral clique em Build → Realtime Database → Criar banco de dados → escolha a região us-central1 → modo Iniciar no modo de teste
4
No menu lateral clique em Configurações do projeto (⚙) → aba "Seus apps" → clique no ícone </> para adicionar app web → registre sem Hosting
5
O Firebase vai mostrar um bloco de código. Copie apenas os 3 valores abaixo e cole nos campos:
apiKey: "AIza..." databaseURL: "https://seu-projeto-default-rtdb.firebaseio.com" projectId: "seu-projeto"
